# Phishtopia v2.0.0 — V2 Foundation

Released: 2026-06-22

Commit deployed: `47b19e613d6c3d062df9f2c1fbfc1fe1ab471e48`

## Summary

This release stabilizes the Phishtopia v2 foundation. The major refactor branch was merged into `main`, deployed to the production GCP VM, restarted through PM2, and verified with a live `/health` check.

## Highlights

- Refactored the app into a cleaner `src/` structure.
- Added dedicated layers for routes, controllers, services, middleware, and database queries.
- Split authentication and YouList logic out of the old large route files.
- Added email verification support and a development-friendly verification page.
- Switched password hashing from native `bcrypt` to `bcryptjs` for better Android/Termux/Codespaces compatibility.
- Polished the homepage layout.
- Polished the projects page mobile and desktop layout.
- Polished auth screens.
- Polished YouList desktop card sizing and preserved mobile stacking.
- Confirmed local route smoke tests before merge.
- Created a backup branch before merging the refactor into `main`.
- Confirmed production deploy completion and live health check.

## Notes

The database remains local-only on the production VM. Development access from Codespaces or Termux should use an SSH tunnel instead of exposing PostgreSQL publicly.

Dependency audit warnings are intentionally left for a separate maintenance pass instead of being fixed during the release merge.
